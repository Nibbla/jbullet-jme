public abstract class TriMeshWireframe
extends com.jme.scene.TriMesh

A customised version of TriMesh for use with the PhysicsDebug view.

The aim is to provide encapsulation for the creation of a wireframe from a bullet physics broadphase rigid body, managing the translation/rotation and the rendering using the standard JME renderer.
